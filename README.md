# My Portfolio

This is a portfolio website built using React and Tailwind CSS. It showcases my skills, projects, and provides a way to contact me.

## Installation

1. Clone the repository:

```bash
git clone https://github.com/your-username/portfolio.git
```

2. Install the dependencies:

```bash
cd portfolio
npm install
```

3. Start the development server:

```bash
npm start
```

4. Open your browser and visit [http://localhost:3000](http://localhost:3000) to see the portfolio.

## Features

1. Home: Introduces myself and provides a brief overview.
2. About: Provides more detailed information about my background, skills, and experience.
3. Navbar: Displays the navigation bar for easy navigation between different sections.
4. Skills: Displays my technical skills and areas of expertise.
5. Work: Showcases my recent projects with links to their live demos and source code repositories.
6. Contact: Allows visitors to get in touch with me by filling out a contact form.

## File Structure

- `src/App.js`: The main component that renders all the other components.
- `src/components/Home.jsx`: Introduces myself and provides an overview.
- `src/components/About.jsx`: Displays information about me.
- `src/components/Navbar.jsx`: Displays the navigation bar.
- `src/components/Skills.jsx`: Displays my technical skills.
- `src/components/Work.jsx`: Showcases my recent projects.
- `src/components/Contact.jsx`: Provides a contact form.

## Dependencies

- React: A JavaScript library for building user interfaces.
- Tailwind CSS: A utility-first CSS framework for rapidly building custom designs.

## Customization

To customize the portfolio with your own content, you can modify the following files:

- Update the content in `src/components/Home.jsx` to introduce yourself and provide an overview.
- Add your own information in `src/components/About.jsx`.
- Customize the navigation links in `src/components/Navbar.jsx`.
- Update your skills and expertise in `src/components/Skills.jsx`.
- Add your projects in `src/components/Work.jsx` by modifying the `project` array.
- Customize the contact form in `src/components/Contact.jsx` to suit your needs.
